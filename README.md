# chrome-navigation

## 介绍

	The navigation extensions for chrome instead of Baidu's Home Page! 
	替代百度首页的导航!
	
## 使用
1. chrome浏览器打开扩展程序
2. 首先勾选开发者模式
3. 加载已解压的扩展程序(没钱不能上架)
4. 选择本文件夹即可
![](img/1.png)
5. 打开之后的效果
![](img/2.png)
6. 百度首页导航
![](img/3.png)
